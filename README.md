# my-desktop-dotfiles
bspwm and sxhkd dotfiles.

INSTALLING

BSPWM:

apt-get update && apt update
apt install bspwm 

https://dev.to/jdeepd/bspwm-installation-on-ubuntu-20-04-updated-2021-3kip

FOR WALLPAPER:

apt install feh

FOR TRANSPARENCY:

apt install compton

FOR TAKE A SCREENSHOT:

apt install maim

FOR TERMINAL:

apt install gnome-terminal

FOR CONSOLE FILE MANAGER:

apt install ranger
